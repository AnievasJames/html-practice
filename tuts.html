1 HTML Basics - The Skeleton üíÄ

Think of HTML as the bones of your website! It creates the structure.

Every HTML page starts with this basic structure:

<!DOCTYPE html> <html> <head> <title>My Portfolio</title> </head> <body> <!-- Your content goes here --> </body> </html>
What's happening here?
- <html> = The wrapper for everything
- <head> = Info about your page (like the title in browser tab)
- <body> = What people actually see on the page

2 Creating the Left Sidebar üë§

Let's add your profile section! We'll use a <div> (think of it as a container box).

<div class="sidebar"> <img src="your-photo.jpg" alt="Phil Choi"> <h1>Phil Choi</h1> <h2>Software Developer</h2> <p>My name is Phil Choi and am a passionate software developer. I like to learn new things and build cool stuff.</p> <h3>Contact me:</h3> <p>Email: letphil.contact@gmail.com</p> <p>LinkedIn: linkedin.com/in/yourname</p> </div>
HTML Tags You Just Learned:
- <h1> = Big heading (your name)
- <h2> = Medium heading (your title)
- <p> = Paragraph (regular text)
- <img> = Image (your photo!)

3 Creating Lists üìù

Lists are perfect for projects, skills, and experience!

<div class="main-content"> <h2>Projects</h2> <ul> <li>Project 1</li> <li>Project 2</li> <li>Project 3</li> </ul> <h2>Skills</h2> <ul> <li>Skill 1</li> <li>Skill 2</li> <li>Skill 3</li> </ul> </div>
<ul> = Unordered List (bullets)
<li> = List Item (each bullet point)

4 CSS - Making It Pretty! üé®

CSS is like the clothing and makeup for your HTML skeleton!

Add this in your <head> section:

<style> body { display: flex; font-family: 'Comic Sans MS', sans-serif; margin: 0; padding: 0; } </style>
‚ú® What this does:
- display: flex = Makes items sit side by side
- font-family = Changes the font to Comic Sans
- margin: 0 = Removes space around the edges

Let's style that left sidebar to look like Phil's!

.sidebar { background-color: #f0e6d2; width: 300px; padding: 30px; border-right: 3px solid #333; } .sidebar img { width: 150px; height: 150px; border-radius: 50%; display: block; margin: 0 auto 20px; } .sidebar h1 { font-size: 24px; margin-bottom: 5px; } .sidebar h2 { font-size: 18px; font-weight: normal; margin-bottom: 20px; }
Cool CSS Tricks:

- padding: 30px = Adds space inside the box
- background-color = That cream/beige color

6 Styling the Main Content üìÑ
